# Plan Generator Skill

Read all items in the vault Inbox and Needs_Action folders, analyze them, and generate a prioritized Plan.md.

## How to Generate a Plan

1. **Scan Inbox**: Read all files in `AI_Employee_Vault/Inbox/`
2. **Scan Needs_Action**: Read all files in `AI_Employee_Vault/Needs_Action/`
3. **Categorize**: Group items by type (email, file, WhatsApp, task)
4. **Prioritize**: Sort by priority (high > normal > low) and age (oldest first)
5. **Write Plan**: Create `AI_Employee_Vault/Plan.md` with structured output

## Plan.md Format
```markdown
# Plan

**Generated:** YYYY-MM-DD HH:MM:SS
**Items Analyzed:** N

## High Priority
- [ ] Item description (source: type, from: sender)

## Medium Priority
- [ ] Item description

## Low Priority
- [ ] Item description

## Summary
- Total items: N
- High priority: N
- Requires response: N
- Informational: N

---
*Generated by AI Employee Plan Generator*
```

## Priority Rules
- Emails from clients/VIPs: High
- Messages with "urgent", "asap", "help": High
- File alerts: Medium
- Newsletters: Low
- Mock/test items: Low

## When to Regenerate
- After new items arrive in Inbox
- At the start of each day
- When user requests `/plan-generator`
