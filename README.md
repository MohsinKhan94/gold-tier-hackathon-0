# 🤖 Personal AI Employee - Bronze Tier

An autonomous AI agent that monitors files 24/7, creates alerts, and manages an Obsidian knowledge base using Agent Skills.

## 🎯 Features

- **24/7 File Monitoring** - Automatically detects new files in watched folder
- **7 Agent Skills** - Read, write, organize, and manage vault operations
- **Auto-Updating Dashboard** - Live statistics in Obsidian
- **Local-First Architecture** - Complete privacy, no cloud dependencies
- **Gemini API Integration** - AI-powered reasoning and analysis

## 🛠️ Tech Stack

- **Python 3.13** - Core automation scripts
- **UV** - Fast Python package manager
- **Obsidian** - Local knowledge base & dashboard
- **Gemini API** - AI reasoning engine
- **Watchdog** - File system event monitoring
